git clone: リモートリポジトリから完全なコピーをローカルに作成する。
git branch: 新しいブランチを作成する。
git checkout: 別のブランチに切り替える。
git log: そのリポジトリのコミット履歴を表示する。
git status: 現在のリポジトリの状態を確認する(変更ファイルなど)。
git add: 変更したファイルをステージングエリアに追加する。
git commit: ステージングされた変更をリポジトリに確定する。
git diff: 最新コミットとの差分を表示する。
git push: ローカルのコミットをリモートリポジトリに反映する。
git pull: リモートの変更をローカルリポジトリに取り込む。
git fetch: リモートの情報を取得するが、マージはしない。
git merge: 別ブランチの変更をマージする。
